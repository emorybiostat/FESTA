Installation Instructions. If required, create a sub-directory. Download the FESTA package that suits your OS into that directory. Unpack the FESTA package in the directory. The FESTA package includes example files, the manual and the executable. The executable files is called FESTA. For convenience you may wish to add FESTA to the executable PATH of your operating system.
You may want to use FUGUE (Recommended), for generating the input files for FESTA.
